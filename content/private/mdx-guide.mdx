Some notes besides MDX formatting:

## PARA Format

- Project: Short term efforts in work
- Area: Long term responsibility to manage
- Resource: Topics that may be useful in the future
- Archive: Inactive items outside of the three categories

## Image Grid

<ImageGrid
  columns={3}
  images={[
    { src: "/images/opengraph-image.png", alt: "Photo1", href: "#" },
    { src: "/images/opengraph-image.png", alt: "Photo2", href: "#" },
    { src: "/images/opengraph-image.png", alt: "Photo3" },
  ]}
/>

```tsx
<ImageGrid
  columns={3} // Accepts 2, 3, or 4 columns
  images={[
    { src: "/opengraph-image.png", alt: "Photo1", href: "#" }, // 'href' is optional
    { src: "/photos/photo2.jpg", alt: "Photo2", href: "#" },
    { src: "/photos/photo3.jpg", alt: "Photo3" },
  ]}
/>
```
